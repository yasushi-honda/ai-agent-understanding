# LLMの基礎

## LLMとは
Large Language Model（大規模言語モデル）。膨大なテキストデータで訓練された、自然言語を理解・生成するAIモデル。

## AIエージェントにおけるLLMの役割
- **脳**: 判断する・選択する（Thinking & Decision）
- ユーザーの意図を汲み取る
- どのツール（Function）を呼ぶべきか判断する
- 結果を解釈して次のアクションを決定する

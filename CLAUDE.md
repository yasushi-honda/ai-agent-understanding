# AIエージェント理解 - プロジェクト設定

## プロジェクト概要
AIエージェントの研究・理解・実践のためのワークスペース。

## ルール
- ドキュメントは日本語で作成（技術用語は原文のまま）
- docs/concepts/ のファイルは番号順序を維持
- 図解素材は docs/diagrams/ 配下で管理
- MCPサーバー実装は mcp-servers/ 配下
- 実験コードは experiments/ 配下（気軽に試せる場）
- 外部情報の引用時は出典URLを必ず記載
- ベストプラクティスは調査日を明記（情報の鮮度が重要）

## 技術スタック
- MCP SDK: TypeScript (@modelcontextprotocol/sdk)
- ランタイム: Node.js
- クラウド: GCP
